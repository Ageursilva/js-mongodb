Api Devs do Futuro

Api de teste de integração com o MongoDB do projeto Devs do Futuro

GET /devs

Retorna todos os desenvolvedores cadastrados.

GET http://localhost:3000/devs

GET /devs/:name

Retorna um desenvolvedor específico pelo nome.

GET http://localhost:3000/devs/{name}

POST /devs

Cadastra um novo desenvolvedor. Envie um JSON com os campos `name` e `age`.

POST http://localhost:3000/devs 
{
  "name": "Nome do Dev",
  "age": 25
}

PUT /devs/:name

Atualiza a idade de um desenvolvedor pelo nome. Envie o campo `age` no corpo da requisição.

PUT http://localhost:3000/devs/{name} 
{
  "age": 30
}

DELETE /devs/:name

Deleta um desenvolvedor pelo nome.

DELETE http://localhost:3000/devs/{name}
